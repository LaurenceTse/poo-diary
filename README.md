# 噗噗日记

轻量级排便监测产品高保真可交互原型。面向手机一屏体验，可延伸至微信小程序 / App / Apple Watch。

## 快速预览

用任意静态服务器打开，或直接双击 `index.html`：

```bash
# Python
python3 -m http.server 5173

# 然后访问 http://localhost:5173
```

手机访问同一局域网地址即可真机体验。

## 产品结构

| Tab | 作用 |
| --- | --- |
| 今天 | 未记录 → 引导打卡；已记录 → 健康提示 / 再记一笔 / 分享图 |
| 日历 | 便便打卡日历，多笔显示 ×N，点选回看 / 补记 |
| 我的 | 本周小报、重记今天、清空记录、Watch 预告 |

### 记录分层

1. **是否噗了**（必填）
2. **形态三卦**（必填）：顺滑毕业 / 坚如磐石 / 洪水预警
3. **身体感受**（可选）：含剧痛/见血红旗，切换认真提醒
4. **一日多记**：已送达后可「再记一笔」，次数越高反馈越夸张

### 情绪反馈

每一步全屏变色 + 夸张网感文案；二连及以上叠加震动/闪白/脉冲；红旗症状关闭玩笑语气并提示就医。

### 分享图

肚肚小报生成海报：只晒「称号」，不晒具体次数；支持保存到相册 / 系统分享。

### 数据

保存在浏览器本地，可在「我的」清空。旧版数据会自动迁移。

## 设计原则

- 一屏完成：主路径无需滚动
- 强反馈：动效 + 文案同时给情绪价值
- 提醒式诊断：轻松、不吓人、不替代医疗
- 成人向卡通：emoji + 手绘感字体，但信息层级清晰
